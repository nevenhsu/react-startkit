# React Initial Project
--

#### Dependencies
- *react*
- *react-dom*
- *react-redux*
- *redux*
- *redux-thunk*
- *axios*
- *bootstrap*
- *@material-ui/core*
- *jss*

#### devDependencies
- *webpack*
- *babel-loader*
- *babel/core*
- *babel/preset-env*
- *babel/preset-react*

### How to run dev mode
`npm install` (First time)
`npm run dev`

### How to run https mode
`npm run https`

### How to build
`npm run build`

### How to run prod mode
`npm run start`
